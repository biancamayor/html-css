Principais formatações:

Negrito/destaque:

Podemos tornar o texto em negrito usando duas formas:

Com a tag B temos um negrito obsoleto.

Já, com a tag STRONG, temos o negrito utilizado atualmente.

Itálico/Ênfase:

Também temos dois métodos:

Com a tag I temos um itálico desatualizado.

Já, com a tag EM, temos um itálico utilizado atualmente.

Texto marcado

Podemos também criar um texto marcado utilizando a tag MARK

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto incluído

Podemos marcar um texto como incluído para indicar que devemos dar ênfase para ele ou que ele foi adicionado depois.

Texto sobrescrito:

Para escrever coisas do tipo: X20 + 3

Texto subscrito:

Para escrever coisas como: H2O
